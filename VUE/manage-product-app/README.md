# 🛍️ Manage Product App (상품 관리 시스템)

Vue 3 + Vite 기반으로 제작한 **상품 관리 웹 애플리케이션**입니다.  
관리자 등록 → 상품 추가 → 수정 / 삭제 → 재고 조정 → 검색·필터·정렬까지  
**상품 관리의 전체 흐름**을 컴포넌트 단위로 구현했습니다.

---

## 📌 주요 기능

### 1. 관리자 등록

- 관리자 이름을 입력해야 상품 등록 가능
- 관리자 미등록 시 상품 추가 차단 (alert 안내)

### 2. 상품 등록

- **간단 모드**: 상품명만 입력
- **상세 모드**:
  - 카테고리
  - 가격
  - 재고
  - 설명
- 상세 모드에서 입력이 있을 경우에만 유효성 검사 적용
- 기본값 자동 처리 (카테고리: 기타, 가격/재고: 0)

### 3. 상품 목록

- 카드(Card) 형태로 상품 표시
- 상품 정보:
  - 이름
  - 카테고리
  - 가격
  - 재고
  - 설명
  - 등록일 / 수정일
- 상품 개수가 많아질 경우 **가로 스크롤 UI 적용**

### 4. 상품 수정 / 삭제

- 수정 버튼 클릭 시:
  - 상품명 직접 수정
  - 기존 데이터를 ProductForm에 불러와 수정
- 저장 시 현재 시간 기준으로 수정일 갱신
- 삭제 버튼으로 상품 제거

### 5. 재고 조정

- 재고 변경량 입력 후 적용
- 입고(+10) 버튼 제공
- 재고가 음수가 될 경우 alert로 차단

### 6. 검색 / 필터 / 정렬

- 검색: 상품명 / 설명 기준
- 카테고리 필터:
  - 전체 / 전자기기 / 의류 / 식품 / 도서 / 기타
- 정렬:
  - 최신순 (기본값)
  - 오래된순
  - 이름순
  - 가격순
  - 재고순
- 현재 선택된 필터/정렬 버튼은 활성 스타일 적용

### 7. 상품 현황 요약

- 전체 상품 개수
- 전체 상품 총 금액 (가격 × 재고)
- `computed`로 실시간 계산

---

## 🧱 기술 스택

- **Frontend**
  - Vue 3 (Composition API)
  - Vite
- **State 관리**
  - `ref`, `reactive`
  - `computed`, `watch`
- **스타일링**
  - CSS (px 기반 고정 레이아웃)
- **기타**
  - UUID 생성: `crypto.randomUUID()`

## 📁 프로젝트 구조

```
manage-product-app
├─ src
│ ├─ components
│ │ ├─ AddProduct.vue # 상품 추가 (간단/상세 모드)
│ │ ├─ CountProduct.vue # 상품 개수 / 총 금액 표시
│ │ ├─ JoinManager.vue # 관리자 등록
│ │ ├─ ProductCard.vue # 상품 카드 (수정/삭제/재고 조정)
│ │ ├─ ProductFilter.vue # 검색 / 카테고리 / 정렬 필터
│ │ ├─ ProductForm.vue # 상품 상세 입력 폼
│ │ └─ ProductList.vue # 상품 목록 + 가로 스크롤
│ ├─ utils
│ │ └─ formValidator.js # 상품 유효성 검사 로직
│ ├─ App.vue # 메인 컨테이너 (상태 관리)
```
