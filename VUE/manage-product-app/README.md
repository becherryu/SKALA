# 📦 상품 관리 시스템 (Manage Product App)

Vue 3 기반의 **관리자용 상품 관리 및 현황 대시보드** 웹 애플리케이션입니다.  
상품 등록/수정/삭제부터 카테고리별 통계, 재고 부족 상품 모니터링까지  
실무 관리자 페이지 구조를 목표로 구현했습니다.

---

## 🚀 주요 기능

### ✅ 관리자 인증

- 관리자 이름 기반 간단 로그인
- 로그인 상태 유지
- 로그인 상태에 따라 기능 접근 제어

### ✅ 상품 관리

- 상품 등록 / 수정 / 삭제
- 카테고리, 가격, 재고, 설명 관리
- 상품 목록 필터링 및 정렬
- 페이지네이션 적용

### ✅ 상품 현황 대시보드

- 전체 상품 개수
- 전체 재고 금액
- 카테고리별 상품 수 차트
- **재고 부족 상품 현황 테이블**
  - 재고 ≤ 5 상품 필터
  - 재고 적은 순 정렬 (품절 우선)
  - 테이블 헤더 고정
  - 테이블 영역만 세로 스크롤

---

## 🧩 기술 스택

- **Frontend**
  - Vue 3 (Composition API)
  - Vite
  - Pinia (인증 상태 관리)
  - Vue Router
  - Chart.js + vue-chartjs

- **Backend (Mock)**
  - json-server

---

## 📁 프로젝트 구조

```
.
public
src
├─ api # API 통신
├─ components # UI 컴포넌트
│ ├─ Home # 상품 관리 화면
│ ├─ About # 상품 현황 대시보드
│ └─ common # 공통 컴포넌트
├─ composables # 비즈니스 로직 (hooks)
├─ pages # 라우팅 페이지
├─ router # 라우터 설정
├─ stores # 전역 상태 (Pinia)
└─ layouts # 레이아웃
```
